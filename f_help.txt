`ping`
Returns a very helpful message about your ping.

`info`
Returns the most important information about Olaf.

`uptime`
Returns the amount of time Olaf has been online for.

`timer [hh:mm:ss|mm:ss|ss] (text)`
Sets a timer for x amount of time with a description.

`quote`
Returns a random quote from #quotes.

`insult`
Returns an extravagant insult for you to use.

`fun`, `fact`, `funfact`
Returns a random fun fact.
